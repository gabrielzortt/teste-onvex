01

Introdução

A presente Política de Privacidade descreve como a plataforma Onvex, operada por [NOME DA EMPRESA / CNPJ], coleta, utiliza, armazena e protege os dados pessoais dos usuários, em conformidade com a Lei nº 13.709/2018 (LGPD).

Ao utilizar a plataforma, o usuário concorda com esta política.

02

Dados coletados

Podemos coletar os seguintes dados:

  • Dados cadastrais: nome completo, CPF ou CNPJ, endereço, e-mail e telefone.
  • Dados de uso: IP, localização aproximada, páginas acessadas e tempo de uso.
  • Dados de pagamento: informações necessárias para transações e dados processados por intermediadores, como a Stone.

A plataforma não armazena dados completos de cartão de crédito.

03

Finalidade do uso dos dados

  • Processar pedidos.
  • Realizar entregas.
  • Efetuar pagamentos.
  • Comunicar usuários.
  • Prevenir fraudes.
  • Cumprir obrigações legais.
04

Compartilhamento de dados

Seus dados podem ser compartilhados com:

  • Produtores.
  • Parceiros logísticos.
  • Intermediadores de pagamento, como a Stone.
  • Autoridades legais, quando exigido.

Nunca vendemos dados para terceiros.

05

Segurança das informações

  • Criptografia de ponta a ponta.
  • Controle de acesso.
  • Monitoramento de segurança.

Pagamentos são processados por sistemas certificados, seguindo padrões como PCI DSS.

06

Armazenamento dos dados

Os dados são mantidos pelo tempo necessário para cumprir obrigações legais e manter o funcionamento da plataforma.

Após isso, os dados podem ser excluídos ou anonimizados.
